# Lumina — Navigacija Duše

This repository contains a single-page HTML app (Lumina). Use GitHub Pages to publish it as a static site.

Quick publish steps (web UI)
1. Create a new repository on GitHub: https://github.com/new  
   - Repository name suggestion: `lumina`  
   - Visibility: Public (recommended)  
2. Put the files from this folder into the repo root and push (or upload via the GitHub web UI).  
3. On GitHub, open the repository → Settings → Pages.  
   - Under "Source" choose branch `main` and folder `/ (root)` → Save.  
4. Wait ~1 minute. Your site will be available at:
   `https://<your-username>.github.io/<repo-name>/`  
5. Optionally enable "Enforce HTTPS" in the Pages settings.

ZIP contents:
- index.html
- README.md
- (optional) package.json and server.js if you want to run locally with Node
